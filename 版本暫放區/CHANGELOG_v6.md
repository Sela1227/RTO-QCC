# Changelog

本檔案記錄所有版本變更。

格式基於 [Keep a Changelog](https://keepachangelog.com/)，
版本號遵循 [Semantic Versioning](https://semver.org/)。

## [Unreleased]

## [2.0.0] - 2024-12-24

### Added
- 追蹤清單新增「無法量」篩選標籤
- 無法測量病人顯示狀態提示，記錄體重後自動恢復追蹤
- 警示規則管理改為點擊編輯模式
- 新增病人對話框優化（生日 8 碼輸入、治療目的選單）
- 舊病人開新療程表單（只需填療程資料，病人資料自動帶入）

### Changed
- 版本號升級至 2.0.0
- 追蹤清單改為主從視圖（左側卡片列表 + 右側詳情）
- 首頁搜尋改為按鈕觸發，支援部分匹配
- 體重記錄錯誤訊息更精確顯示
- **暫停中療程視為未結束，不能開新療程**（需先結案或終止）

### Fixed
- **修復：恢復療程按鈕無法作用**（缺少 resume 方法）
- **修復：暫停中病人可以開新療程**（應該只有已結束才能開新療程）
- 修復：新增病人無法儲存（方法名稱錯誤）
- 修復：無法測量病人無法記錄體重（錯誤處理問題）
- 修復：報表 SQL ambiguous column 錯誤
- 修復：首頁統計數字不一致問題

## [1.2.0] - 2024-12-19

### Added
- 體重記錄可修改/刪除，支援更改日期
- 營養轉介單 PDF 生成功能（癌症資源中心轉介單格式）
- 警示規則管理（支援不同癌別設定不同閾值）
- 文件管理區（上傳 SDM 單張、衛教單張等）

### Changed
- 體重記錄列表支援點擊編輯
- 警示等級判斷支援癌別特定規則

## [1.1.0] - 2024-12-19

### Added
- 病人資料新增「治療目的」欄位（治癒性/姑息性/其他）
- 病人資料新增「生日」必填欄位，自動計算並顯示年紀
- 設定頁新增癌別管理功能（新增/刪除/排序）

### Changed
- 病歷號固定 7 碼，不足前面補 0
- 所有頁面顯示 7 碼病歷號格式
- 追蹤清單、療程詳情、病人清單顯示年紀

### Fixed
- **啟動時自動修復舊數據的體重變化率**（確保下降為負、上升為正）
- 體重變化率計算邏輯確認正確（下降為負，上升為正）

## [1.0.0] - 2024-12-18

### Added

- 病人管理：新增、搜尋、編輯、刪除（含 4 碼確認）
- 療程管理：建立療程、暫停、恢復、終止、結案（含 4 碼確認）
- 體重記錄：新增體重、自動計算變化率、趨勢圖表
- 介入管理：SDM、營養師轉介、手動介入、略過功能
- 自動警示：體重下降達閾值自動觸發介入
- 追蹤清單：依追蹤狀態分類、逾期/待量測提醒
- 無法測量：療程可標記為無法測量，不計入逾期
- 統計報表：
  - 基本統計（新病人、新療程、量體重、結案）
  - 體重變化分布圖（下降≥5%、3-5%、維持）
  - 介入執行率（SDM、營養師）
  - 追蹤品質（按時量測率、平均間隔）
  - 癌別分析（人數、平均變化率、介入率）
- 報表篩選：依癌別、狀態、排除無法測量
- Excel 匯出：完整統計 + 明細資料
- 系統設定：警示閾值、逾期天數可調整
