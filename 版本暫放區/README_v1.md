# 放射線治療體重追蹤系統

Radio-Oncology Weight Tracking System

## 功能

- 病人管理（Barcode 掃描）
- 療程追蹤（多療程支援）
- 體重監控（自動警示 3%/5%）
- 介入記錄（SDM / 營養師轉介）
- 統計報表（Excel 匯出）

## 安裝

```bash
pip install -r requirements.txt
```

## 執行

```bash
python main.py
```

## 技術

- Python 3.10+
- Flet (UI Framework)
- SQLite (Database)
- SELA Design System

## 版本

- 1.0.0 - 初版
