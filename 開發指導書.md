## 1. 系統概述

### 1.1 目標

追蹤放射線治療病人在治療期間的體重變化，根據體重下降比例自動提示醫療介入：

| 體重下降比例 | 介入行動 |
|-------------|---------|
| ≥ 3% | 觸發 SDM（共享決策）提醒 |
| ≥ 5% | 建議轉介營養師 |

### 1.2 使用情境

- 每日約 50 位病人需要量測體重
- 每月約 50 位新病人
- 單機使用，未來可能擴展至多機或整合 HIS

### 1.3 核心概念

```
病人 (Patient)           療程 (Treatment)           體重/介入記錄
┌─────────────┐         ┌─────────────┐          ┌─────────────┐
│ 病歷號(唯一) │   1:N   │ 癌別        │    1:N   │ 體重記錄    │
│ 姓名        │ ──────► │ 治療期間    │ ───────► │ 介入記錄    │
│ 性別        │         │ 基準體重    │          │             │
└─────────────┘         │ 狀態        │          └─────────────┘
                        └─────────────┘

- 同一病人可有多次療程（如：2024 頭頸癌、2025 肺癌）
- 每次療程獨立追蹤體重變化
- 病歷號為唯一識別，判斷是否同一人
```

### 1.4 設計原則

| 原則 | 說明 |
|------|------|
| 簡化輸入 | 預設值自動帶入，減少手動輸入 |
| 自動計算 | 變化率、警示、療程名稱自動產生 |
| 隱藏不常用 | 進階功能收到選單，保持介面簡潔 |
| 資料安全 | 自動備份、版本升級相容 |
| 可擴展 | 預留 HIS 整合介面、JSON 擴展欄位 |

### 3.3 完整 SQL Schema

```sql
-- ============================================================
-- 放射線治療體重追蹤系統 - 資料庫 Schema
-- 版本: 1.0.0
-- ============================================================

-- 1. 病人表
CREATE TABLE patients (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    medical_id      VARCHAR(50) UNIQUE NOT NULL,
    name            VARCHAR(100) NOT NULL,
    gender          VARCHAR(10),
    birth_date      DATE,
    extra_fields    JSON,
    created_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 2. 療程表
CREATE TABLE treatments (
    id                  INTEGER PRIMARY KEY AUTOINCREMENT,
    patient_id          INTEGER NOT NULL REFERENCES patients(id),
    cancer_type         VARCHAR(100) NOT NULL,
    treatment_name      VARCHAR(255),
    treatment_start     DATE NOT NULL,
    treatment_end       DATE,
    baseline_weight     DECIMAL(5,2) NOT NULL,
    baseline_date       DATE NOT NULL,
    status              VARCHAR(20) DEFAULT 'active',
    status_reason       VARCHAR(255),
    status_changed_at   TIMESTAMP,
    extra_fields        JSON,
    created_at          TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at          TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 3. 體重記錄表
CREATE TABLE weight_records (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    treatment_id    INTEGER NOT NULL REFERENCES treatments(id),
    measure_date    DATE NOT NULL,
    weight          DECIMAL(5,2) NOT NULL,
    change_rate     DECIMAL(5,2),
    alert_level     VARCHAR(20),
    notes           TEXT,
    created_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 4. 介入記錄表
CREATE TABLE interventions (
    id                  INTEGER PRIMARY KEY AUTOINCREMENT,
    treatment_id        INTEGER NOT NULL REFERENCES treatments(id),
    weight_record_id    INTEGER REFERENCES weight_records(id),
    type                VARCHAR(20) NOT NULL,
    source              VARCHAR(20) DEFAULT 'auto',
    trigger_reason      TEXT,
    status              VARCHAR(20) DEFAULT 'pending',
    executed_at         DATE,
    executed_by         VARCHAR(100),
    result              TEXT,
    skip_reason         VARCHAR(50),
    skip_note           TEXT,
    created_at          TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at          TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 5. 系統設定表
CREATE TABLE settings (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    category        VARCHAR(50) NOT NULL,
    key             VARCHAR(100) NOT NULL,
    value           TEXT,
    value_type      VARCHAR(20),
    description     VARCHAR(255),
    updated_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    UNIQUE(category, key)
);

-- 6. 療程狀態歷史表
CREATE TABLE treatment_status_history (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    treatment_id    INTEGER NOT NULL REFERENCES treatments(id),
    from_status     VARCHAR(20),
    to_status       VARCHAR(20),
    reason          VARCHAR(255),
    note            TEXT,
    changed_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    changed_by      VARCHAR(100)
);

-- 7. 資料庫版本表
CREATE TABLE _db_migrations (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    version         VARCHAR(20) NOT NULL,
    migration_name  VARCHAR(100) NOT NULL,
    applied_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    checksum        VARCHAR(64)
);

-- ============================================================
-- 索引
-- ============================================================

CREATE INDEX idx_patient_medical_id ON patients(medical_id);
CREATE INDEX idx_treatment_patient ON treatments(patient_id);
CREATE INDEX idx_treatment_status ON treatments(status);
CREATE INDEX idx_weight_treatment ON weight_records(treatment_id);
CREATE INDEX idx_weight_date ON weight_records(measure_date);
CREATE INDEX idx_intervention_treatment ON interventions(treatment_id);
CREATE INDEX idx_intervention_status ON interventions(status);
CREATE INDEX idx_settings_category_key ON settings(category, key);

-- ============================================================
-- 預設設定值
-- ============================================================

INSERT INTO settings (category, key, value, value_type, description) VALUES
-- 警示設定
('alert', 'sdm_threshold', '3', 'number', 'SDM 提醒閾值 (%)'),
('alert', 'nutrition_threshold', '5', 'number', '營養師轉介閾值 (%)'),

-- 提醒設定
('reminder', 'warning_days', '5', 'number', '幾天後開始提醒'),
('reminder', 'overdue_days', '7', 'number', '幾天後視為逾期'),
('reminder', 'show_on_startup', 'true', 'boolean', '啟動時顯示逾期提醒'),

-- 報表設定
('report', 'default_range', 'this_month', 'string', '預設查詢區間'),
('report', 'page_size', '20', 'number', '每頁顯示筆數'),

-- 系統設定
('system', 'db_version', '1.0.0', 'string', '資料庫版本'),
('system', 'backup_retention_days', '30', 'number', '備份保留天數');

-- ============================================================
-- 初始版本記錄
-- ============================================================

INSERT INTO _db_migrations (version, migration_name) VALUES
('1.0.0', 'initial_schema');

from typing import Optional, List, Any, Type, TypeVar
from app.models.database import Database

T = TypeVar('T')

class BaseRepository:
    """基礎資料存取類別"""
    
    def __init__(self, db: Database):
        self.db = db
    
    def get_by_id(self, table: str, id: int) -> Optional[dict]:
        """依 ID 查詢單筆"""
        sql = f"SELECT * FROM {table} WHERE id = ?"
        return self.db.fetch_one(sql, (id,))
    
    def get_all(self, table: str) -> List[dict]:
        """查詢全部"""
        sql = f"SELECT * FROM {table}"
        return self.db.fetch_all(sql)
    
    def insert(self, table: str, data: dict) -> int:
        """新增一筆，回傳 ID"""
        columns = ", ".join(data.keys())
        placeholders = ", ".join(["?"] * len(data))
        sql = f"INSERT INTO {table} ({columns}) VALUES ({placeholders})"
        return self.db.execute(sql, tuple(data.values()))
    
    def update(self, table: str, id: int, data: dict) -> bool:
        """更新一筆"""
        set_clause = ", ".join([f"{k} = ?" for k in data.keys()])
        sql = f"UPDATE {table} SET {set_clause}, updated_at = CURRENT_TIMESTAMP WHERE id = ?"
        self.db.execute(sql, tuple(data.values()) + (id,))
        return True
    
    def delete(self, table: str, id: int) -> bool:
        """刪除一筆"""
        sql = f"DELETE FROM {table} WHERE id = ?"
        self.db.execute(sql, (id,))
        return True
    
    def count(self, table: str, where: str = None, params: tuple = None) -> int:
        """計數"""
        sql = f"SELECT COUNT(*) as count FROM {table}"
        if where:
            sql += f" WHERE {where}"
        result = self.db.fetch_one(sql, params or ())
        return result["count"] if result else 0

            JOIN treatments t ON w.treatment_id = t.id
            JOIN patients p ON t.patient_id = p.id
            WHERE w.measure_date BETWEEN ? AND ?
        """

        rows = self.db.fetch_all(sql, tuple(params))
        
        return {

            "total": total,
            "page": page,

from typing import Optional, List
from datetime import date
from app.models.weight_record import WeightRecord
from app.repositories.weight_repository import WeightRepository
from app.repositories.treatment_repository import TreatmentRepository
from app.services.intervention_service import InterventionService

class WeightService:
    """體重記錄服務"""
    
    def __init__(self, weight_repo: WeightRepository,
                 treatment_repo: TreatmentRepository,
                 intervention_service: InterventionService):
        self.weight_repo = weight_repo
        self.treatment_repo = treatment_repo
        self.intervention_service = intervention_service
    
    def create(self, treatment_id: int, weight: float, 
               measure_date: date = None, notes: str = None) -> WeightRecord:
        """新增體重記錄"""
        # 預設日期為今天
        if measure_date is None:
            measure_date = date.today()
        
        # 檢查同日期是否已有記錄
        existing = self.weight_repo.get_by_date(treatment_id, measure_date)
        if existing:
            raise ValueError(f"該日期已有記錄（{existing.weight} kg）")
        
        # 計算變化率
        treatment = self.treatment_repo.get_by_id(treatment_id)
        change_rate = self._calculate_change_rate(treatment.baseline_weight, weight)
        
        # 判斷警示等級
        alert_level = self._determine_alert_level(change_rate)
        
        # 建立記錄
        record = WeightRecord(
            treatment_id=treatment_id,
            measure_date=measure_date,
            weight=weight,
            change_rate=change_rate,
            alert_level=alert_level,
            notes=notes,
        )
        record = self.weight_repo.create(record)
        
        # 自動建立介入記錄
        if alert_level in ("sdm", "nutrition"):
            self.intervention_service.create_auto(
                treatment_id=treatment_id,
                weight_record_id=record.id,
                alert_level=alert_level
            )
        
        return record
    
    def get_history(self, treatment_id: int) -> List[WeightRecord]:
        """取得體重歷史"""
        return self.weight_repo.get_by_treatment(treatment_id)
    
    def get_chart_data(self, treatment_id: int) -> dict:
        """取得圖表資料"""
        records = self.weight_repo.get_by_treatment(treatment_id)
        records.reverse()  # 按日期正序
        
        treatment = self.treatment_repo.get_by_id(treatment_id)
        
        return {
            "dates": [r.measure_date.isoformat() for r in records],
            "weights": [r.weight for r in records],
            "baseline": treatment.baseline_weight,
            "sdm_line": treatment.baseline_weight * 0.97,  # -3%
            "nutrition_line": treatment.baseline_weight * 0.95,  # -5%
        }
    
    def _calculate_change_rate(self, baseline: float, current: float) -> float:
        """計算變化率（正值表示體重下降）"""
        if baseline <= 0:
            return 0.0
        rate = (baseline - current) / baseline * 100
        return round(rate, 2)
    
    def _determine_alert_level(self, change_rate: float) -> str:
        """判斷警示等級"""
        from app.services.setting_service import SettingService
        
        # 從設定取得閾值
        settings = SettingService.get_alert_settings()
        nutrition_threshold = settings.get("nutrition_threshold", 5)
        sdm_threshold = settings.get("sdm_threshold", 3)
        
        if change_rate >= nutrition_threshold:
            return "nutrition"
        elif change_rate >= sdm_threshold:
            return "sdm"
        return "none"

from app.repositories.treatment_repository import TreatmentRepository
from app.repositories.patient_repository import PatientRepository
from app.repositories.weight_repository import WeightRepository
from app.repositories.intervention_repository import InterventionRepository
from app.services.setting_service import SettingService

class TrackingService:
    """追蹤狀態服務"""
    
    def __init__(self, treatment_repo: TreatmentRepository,
                 patient_repo: PatientRepository,
                 weight_repo: WeightRepository,
                 intervention_repo: InterventionRepository,
                 setting_service: SettingService):
        self.treatment_repo = treatment_repo
        self.patient_repo = patient_repo
        self.weight_repo = weight_repo
        self.intervention_repo = intervention_repo
        self.setting_service = setting_service
    
    def get_tracking_list(self, status_filter: str = "active",
                          tracking_filter: str = None) -> List[Treatment]:
        """取得追蹤清單"""
        # 依療程狀態查詢
        if status_filter == "active":
            treatments = self.treatment_repo.get_all_active()
        else:
            treatments = self.treatment_repo.get_by_status(status_filter)
        
        # 組裝關聯資料
        for t in treatments:
            t.patient = self.patient_repo.get_by_id(t.patient_id)
            t.last_weight = self.weight_repo.get_latest(t.id)
            t.tracking_status = self._calculate_tracking_status(t)
            t.days_since_last = self._calculate_days_since(t)
            t.pending_interventions = self.intervention_repo.get_pending_by_treatment(t.id)
            
            if t.last_weight:
                t.current_weight = t.last_weight.weight
                t.current_change_rate = t.last_weight.change_rate
        
        # 追蹤狀態篩選
        if tracking_filter and tracking_filter != "all":
            if tracking_filter == "pending_intervention":
                treatments = [t for t in treatments if t.pending_interventions]
            else:
                treatments = [t for t in treatments if t.tracking_status == tracking_filter]
        
        # 排序：逾期 > 待量測 > 正常
        return self._sort_by_priority(treatments)
    
    def get_dashboard_summary(self) -> dict:
        """首頁統計數據"""
        treatments = self.get_tracking_list("active")
        
        return {
            "total_active": len(treatments),
            "overdue_count": sum(1 for t in treatments if t.tracking_status == "overdue"),
            "pending_count": sum(1 for t in treatments if t.tracking_status == "pending"),
            "sdm_pending": self.intervention_repo.count_pending_by_type("sdm"),
            "nutrition_pending": self.intervention_repo.count_pending_by_type("nutrition"),
        }
    
    def get_overdue_list(self) -> List[Treatment]:
        """取得逾期清單（提醒彈窗用）"""
        treatments = self.get_tracking_list("active", "overdue")
        return treatments
    
    def _calculate_tracking_status(self, treatment: Treatment) -> str:
        """計算追蹤狀態"""
        if treatment.status != "active":
            return treatment.status
        
        if not treatment.last_weight:
            return "overdue"
        
        days = self._calculate_days_since(treatment)
        settings = self.setting_service.get_reminder_settings()
        
        overdue_days = settings.get("overdue_days", 7)
        warning_days = settings.get("warning_days", 5)
        
        if days > overdue_days:
            return "overdue"
        elif days >= warning_days:
            return "pending"
        return "normal"
    
    def _calculate_days_since(self, treatment: Treatment) -> int:
        """計算距離上次量測天數"""
        if not treatment.last_weight:
            return 999
        
        last_date = treatment.last_weight.measure_date
        if isinstance(last_date, str):
            last_date = date.fromisoformat(last_date)
        
        return (date.today() - last_date).days
    
    def _sort_by_priority(self, treatments: List[Treatment]) -> List[Treatment]:
        """依優先級排序"""
        priority = {
            "overdue": 0,
            "pending": 1,
            "normal": 2,
            "paused": 3,
            "completed": 4,
            "terminated": 5,
        }
        
        return sorted(treatments, key=lambda t: (
            priority.get(t.tracking_status, 99),
            -t.days_since_last if t.days_since_last else 0
        ))
```

### 13.4 追蹤狀態計算

```python
def calculate_tracking_status(treatment, last_weight_date, settings):
    """
    計算追蹤狀態
    
    Parameters:
    - treatment: 療程資料
    - last_weight_date: 最後量測日期
    - settings: 提醒設定 {warning_days, overdue_days}
    
    Returns:
    - "normal": 正常（7天內有量）
    - "pending": 待量測（5-7天未量）
    - "overdue": 逾期（超過7天未量）
    """
    
    if treatment.status != "active":
        return treatment.status  # paused/terminated/completed
    
    if last_weight_date is None:
        return "overdue"
    
    days_since = (date.today() - last_weight_date).days
    
    if days_since > settings["overdue_days"]:  # 預設 7
        return "overdue"
    elif days_since >= settings["warning_days"]:  # 預設 5
        return "pending"
    else:
        return "normal"
```

## 16. 版本升級機制

### 16.1 版本號規則

```
版本格式：主版本.次版本.修訂版（如 1.2.3）

主版本 ↑  重大變更，可能不相容（需遷移工具）
次版本 ↑  新功能，向下相容
修訂版 ↑  Bug 修復，完全相容
```

### 16.2 Migration 機制

```python
# migrations/versions/v001_initial_schema.py

VERSION = "1.0.0"
DESCRIPTION = "初始資料庫結構"

def upgrade(db):
    """升級：建立初始表格"""
    db.execute("""
        CREATE TABLE patients (...)
    """)
    db.execute("""
        CREATE TABLE treatments (...)
    """)
    # ...

def downgrade(db):
    """降級：刪除表格"""
    db.execute("DROP TABLE IF EXISTS patients")
    # ...

def validate(db):
    """驗證：確認遷移成功"""
    tables = db.execute("SELECT name FROM sqlite_master WHERE type='table'")
    required = ['patients', 'treatments', 'weight_records', 'interventions', 'settings']
    return all(t in tables for t in required)
```

### 16.4 版本相容檢查（匯入時）

```python
def check_import_compatibility(file_version, current_version):
    """
    檢查匯入檔案版本相容性
    
    Returns:
    {
        "compatible": True/False,
        "message": "...",
        "migration_needed": True/False
    }
    """
    file_major = int(file_version.split('.')[0])
    curr_major = int(current_version.split('.')[0])
    
    if file_major > curr_major:
        return {
            "compatible": False,
            "message": "匯入檔案版本較新，請先升級程式",
        }
    
    if file_major < curr_major:
        return {
            "compatible": True,
            "message": "匯入舊版資料，部分新欄位將使用預設值",
            "migration_needed": True,
        }
    
    return {"compatible": True}
```
